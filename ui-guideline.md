UI GUIDELINES FOR THIS PROJECT

GENERAL FEELING:
- Warm
- Family-friendly
- Trustworthy
- Suitable for wedding, home party, baby party services
- Slightly elegant, not luxury-heavy

DESIGN PRINCIPLES:
- Simple layouts
- Clear spacing
- Easy to scan content
- CTA visible without scrolling too much
- No flashy animation

COLOR USAGE RULES:
- Use primary color for:
  - Header background
  - Section titles
  - Icons
- Use accent color ONLY for:
  - CTA buttons
  - Highlight price
  - Important keywords
- Avoid using more than 3 main colors on one screen

TYPOGRAPHY:
- Font style: friendly, readable, Vietnamese-friendly
- Headings: slightly rounded, not sharp
- Body text: normal weight, high readability
- Avoid ultra-thin fonts

SPACING & LAYOUT:
- Section padding (desktop): 80px–100px
- Section padding (mobile): 40px–60px
- Max content width: 1200px
- Center content, avoid full-width text blocks

BUTTON RULES:
- Border radius: medium to large
- CTA buttons must:
  - Stand out clearly
  - Use accent color
  - Be full-width on mobile
- Button text must be action-based:
  - “Đặt tiệc ngay”
  - “Gọi tư vấn”
  - “Xem menu”

IMAGE RULES:
- Images must relate to:
  - Food
  - Party
  - Wedding
  - Family gathering
- Avoid stock images that look corporate
- Always include alt text

ICON RULES:
- Use simple outline or solid icons
- Avoid complex illustrations
- Icons should support content, not dominate

DO NOT:
- Do NOT use SaaS dashboard layouts
- Do NOT use neon or strong contrast colors
- Do NOT overuse shadows
- Do NOT mix too many font sizes
